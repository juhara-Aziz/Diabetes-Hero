# بطل السكر

#"بطل السكري" هو تطبيق تعليمي مخصص للأطفال المصابين بمرض السكري، ويهدف إلى توعيتهم وتعليمهم العادات الصحية اليومية بطريقة تفاعلية وممتعة.

يركز البرنامج على تقديم المعلومات الصحية للأطفال بأسلوب بسيط من خلال مستويات تعليمية تحتوي على أسئلة وأنشطة تساعد الطفل على فهم مرض السكري والتعامل معه بشكل صحيح.

كما يوفر البرنامج واجهة خاصة لولي الأمر تمكنه من متابعة تقدم الطفل ومستوى تعلمه.

## لتقنيات المستخدمة
- google colab
- python
- streamlit
